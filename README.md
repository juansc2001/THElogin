# THElogin
THELogin é uma aplicação Django desenvolvida para servir como base de autenticação em projetos web. O projeto reúne funcionalidades comuns encontradas em sistemas reais, permitindo reutilização de código e servindo como material de estudo para outros desenvolvedores.

## Objetivo
Essa aplicação web tem como objetivo realizar o login dos usuarios com isso traz todas as ferramentas que um sistema de login pode oferecer(ou quase todos). Tela de login e senha, cadastro de novos usuarios, login por conta do google, sistema de recuperação de senha, sistema de pagamento por pix para login premium.

## Funcionalidades

1. [ ] cadastro de novos usuarios
2. [ ] cadastro de usuarios premium(pagamento por pix)
3. [ ] login de usuarios
4. [ ] recuperação de senha
5. [ ] troca de email
6. [ ] apresentar informações importantes quando o usurio estiver logado (nome, email, conta_premium/comum, telefone)

## Tecnologias Utilizadas

1. python
2. django(framework)
3. javascript
4. html + css
